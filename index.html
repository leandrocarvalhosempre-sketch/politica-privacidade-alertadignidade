Política de Privacidade

AlertaDignidade 2.0

O aplicativo AlertaDignidade 2.0 foi desenvolvido para fornecer recursos de segurança pessoal e envio de alertas de emergência.

Dados coletados

O aplicativo pode utilizar:

  • Localização do dispositivo
  • Permissão para envio de SMS
  • Permissão para realizar chamadas telefônicas

Essas permissões são utilizadas exclusivamente para funcionamento das funcionalidades do aplicativo e somente quando acionadas pelo usuário.

Compartilhamento de dados

Não compartilhamos dados pessoais com terceiros.
